Создай современное веб-приложение на Ruby on Rails 8. Это защищенный мессенджер на русском языке.

### Стек технологий:
- Ruby 3.3+, Rails 8.
- База данных: PostgreSQL.
- Frontend: Tailwind CSS, Hotwire (Turbo + Stimulus) для работы в реальном времени без перезагрузок.
- Real-time: Action Cable (используй Solid Cable, стандарт Rails 8).
- Фоновые задачи: Solid Queue.
- Аутентификация: Используй встроенный генератор Rails 8 `rails generate authentication` или Devise.

### Основной функционал:
1. **Аутентификация:** Регистрация/логин по логину и паролю.
2. **Типы сущностей:**
   - Личные чаты (1-на-1).
   - Группы (несколько участников, все могут писать).
   - Каналы (публичные/приватные, пишут только админы).
3. **Шифрование:** - Реализуй симметричное шифрование сообщений на стороне сервера перед сохранением в БД (используй ActiveSupport::MessageEncryptor или встроенное в Rails 7/8 шифрование полей ActiveRecord `encrypts :content`).
4. **Папки и файлы:**
   - Возможность создания папок пользователем.
   - Загрузка изображений в папки (Active Storage).
   - Генерация "зашифрованной ссылки" для доступа к папке (URL с уникальным токеном доступа).
5. **Интерфейс:**
   - Полностью на русском языке.
   - Современный, адаптивный дизайн (темная/светлая темы, лаконичный UI как в Telegram/Signal).
   - Использование компонентов Flowbite или DaisyUI для Tailwind.

### Админ-панель:
- Доступ только для пользователей с ролью `admin`.
- Управление пользователями, просмотр статистики (без доступа к расшифрованным сообщениям).

### Инструкции по выполнению:
1. Создай новую Rails 8 архитектуру.
2. Настрой локализацию (i18n) на русский язык.
3. Реализуй модели: User, Room (polymorphic: chat, group, channel), Message, Folder, Attachment.
4. Настрой контроллеры и представления с использованием Turbo Frames и Turbo Streams для мгновенного обновления сообщений.
5. Напиши тесты для логики шифрования.

Начни с генерации структуры проекта и настройки базовой аутентификации.
